Django Easy Icons

Complete Configuration

Full EASY_ICONS configuration showing all three renderer types. The first renderer ("default") is used when no renderer argument is specified. 


EASY_ICONS = {
    # Default SVG renderer configuration
    "default": {
        "renderer": "easy_icons.renderers.SvgRenderer",
        "config": {
            "svg_dir": "icons",
            "default_attrs": {
                "height": "1em",
                "fill": "currentColor",
            },
        },
        "icons": {
            "home": "home.svg",
            "star": "star.svg",
            "user": "user.svg",
        },
    },

    # Font Awesome provider renderer
    "fontawesome": {
        "renderer": "easy_icons.renderers.ProviderRenderer",
        "config": {
            "tag": "i",
        },
        "icons": {
            "heart": "fas fa-heart",
            "star": "fas fa-star",
            "admin": "fas fa-toolbox",
            "user": "fas fa-user",
            "cog": "fas fa-cog",
        },
    },

    # Bootstrap Icons sprite renderer (using local static file)
    "sprites": {
        "renderer": "easy_icons.renderers.SpritesRenderer",
        "config": {
            "sprite_url": "/static/example/bootstrap-icons.svg",
            "default_attrs": {
                "height": "2em",
                "fill": "currentColor",
            },
        },
        "icons": {
            "zero": "0-circle",
            "one": "1-circle",
            "zero-filled": "0-circle-fill",
            "one-filled": "1-circle-fill",
            "zero-square": "0-square",
        },
    },
}

1. SVG Renderer (Default)

Renders SVG files from the configured directory (default="icons/") with customizable attributes. This is the default renderer, so no renderer argument is needed.

{% verbatim %} {% icon "home" %} {% endverbatim %} {% icon "home" %}
{% verbatim %} {% icon "home" width="32" height="32" class="text-primary" %} {% endverbatim %} {% icon "home" width="32" height="32" class="text-primary" %}

2. Font Awesome Provider Renderer

Renders Font Awesome icons using <i> tags with CSS classes.

Note: Because this is not the default renderer, we must provide the renderer="fontawesome" argument to the template tag.
{% verbatim %} {% icon "heart" renderer="fontawesome" %} {% endverbatim %} {% icon "heart" renderer="fontawesome" %}
{% verbatim %} {% icon "star" renderer="fontawesome" class="text-warning" %} {% endverbatim %} {% icon "star" renderer="fontawesome" class="text-warning" %}
{% verbatim %} {% icon "user" renderer="fontawesome" class="fs-1 text-primary" %} {% endverbatim %} {% icon "user" renderer="fontawesome" class="fs-1 text-primary" %}
{% verbatim %} {% icon "admin" renderer="fontawesome" %} {% endverbatim %} {% icon "admin" renderer="fontawesome" %}

3. SVG Sprites Renderer

References Bootstrap Icons SVG sprite using <svg><use> elements from the local static file: {% static "example/bootstrap-icons.svg" %}

Note: Because this is not the default renderer, we must provide the renderer="sprites" argument to the template tag.
{% verbatim %} {% icon "zero" renderer="sprites" %} {% endverbatim %} {% icon "zero" renderer="sprites" %}
{% verbatim %} {% icon "one" renderer="sprites" width="24" height="24" class="text-success" %} {% endverbatim %} {% icon "one" renderer="sprites" width="24" height="24" class="text-success" %}
{% verbatim %} {% icon "zero-filled" renderer="sprites" class="text-danger" %} {% endverbatim %} {% icon "zero-filled" renderer="sprites" class="text-danger" %}
{% verbatim %} {% icon "zero-square" renderer="sprites" class="text-info" %} {% endverbatim %} {% icon "zero-square" renderer="sprites" class="text-info" %}
